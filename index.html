2. Install Python

  1. Go to the official Python website.
  2. Download the latest version for your OS (Windows, macOS, or Linux).
  3. Run the installer. Check the box that says Add Python to PATH.
  4. Verify the installation by opening a terminal and typing: 
    python --version
    You should see the Python version (e.g., Python 3.11.5).
Note: If you're on macOS or Linux, Python might already be installed. Check using python3 --version.

4. Understand Python Syntax and Best Practices (25 minutes)

Python is known for its simplicity and readability. Here’s a deeper dive into its syntax and best practices:

4.1 Variables and Data Types

Python is dynamically typed, meaning you don’t need to declare variable types. Common data types include:

  • Integers: x = 10
  • Floats: y = 3.14
  • Strings: name = "Python"
  • Booleans: is_fun = True
Tip: Use descriptive variable names for better readability (e.g., user_age instead of ua).

4.2 Indentation and Code Blocks

Python uses indentation (spaces or tabs) to define code blocks. For example:

if 5 > 2:
    print("Five is greater than two!")  # This is part of the if block
print("This is outside the if block")  # This is not
Warning: Mixing tabs and spaces can cause errors. Stick to 4 spaces per indentation level (PEP 8 recommendation).

4.3 Comments

Use # for single-line comments and triple quotes (""") for multi-line comments:

# This is a single-line comment

"""
This is a multi-line comment.
It can span multiple lines.
"""

4.4 Functions

Functions are defined using the def keyword. For example:

def greet(name):
    print(f"Hello, {name}!")  # f-strings make string formatting easy

greet("Alice")  # Output: Hello, Alice!
Tip: Use meaningful function names and keep functions small and focused.

4.5 Lists and Loops

Lists are ordered collections of items. You can loop through them using a for loop:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

4.6 Error Handling

Use try and except to handle errors gracefully:

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

5. Run Your First Python Script (10 minutes)

  1. Open your IDE or text editor and create a file named hello.py.
  2. Add this code: 
    # hello.py
print("Hello, World!")
  3. Save the file and open a terminal.
  4. Navigate to the file's directory and run: 
    python hello.py
  5. You should see: 
    Hello, World!
