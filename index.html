2. Install Python

2.1 Windows

There are two ways for installing Python on your computer with Windows OS:

  1. Using installation media
  2. Install Python in Windows Appstore

2.1.1 Installing Python: USING INSTALLATION MEDIA

Step 1: Download Python
  1. Go to the official Python website: https://www.python.org/downloads/
  2. Click the "Download Python X.X.X" button (it will automatically suggest the latest stable version for Windows).
  3. Save the installer (.exe file) to your computer.
Step 2: Run the Installer
  1. Locate the downloaded file (usually in the Downloads folder) and double-click it to run.
  2. Important: Check the box "Add Python to PATH" at the bottom of the installer window.
  3. Click "Install Now" for a default installation (recommended for most users).
Step 3: Verify the Installation
  1. Open Command Prompt (Win + R, type cmd, and press Enter).
  2. Type the following command and press Enter: 
    python --version
    or 
    python -V
  3. If Python is installed correctly, it will display the installed version (e.g., Python 3.13.2).
Step 4: Install pip (Optional)
  1. pip (Python package manager) is usually included by default. You can check if it's installed by running: 
    pip --version
  2. If it's not installed, you can install it with: 
    python -m ensurepip --default-pip

2.1.2 Installing Python: USING WINDOWS APPSTORE

Step 1: Open Windows Appstore
  1. Open Windows Appstore from your start menu.
  2. Search for Python and click on the first result.
Step 2: Install Python
  1. Click on the install button and wait for the installation to complete.
  2. Open Command Prompt and type python --version to check if Python is installed correctly.

2.2 Unix

Python is pre-installed on most Unix-based systems. You can check the installed version by running:

python --version

If Python is not installed, you can install it using your package manager:

sudo apt-get install python3

Replace apt-get with your package manager (e.g., yum, dnf, zypper) if you're using a different distribution.

Note:
Some Unix-based systems may use python instead of python3 for Python 3.

2.3 macOS

macOS comes with Python 2 pre-installed. You can install Python 3 using Homebrew:

brew install python

After installation, you can run Python 3 using:

python3
Note:
Homebrew is a package manager for macOS. You can install it by running: 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

3. Set Up a Development Environment

Step 1: Install VS Code

Download and install VS Code from: https://code.visualstudio.com/ Open VS Code after installation.

Step 2: Install Essential Extensions

In VS Code, go to Extensions (Ctrl+Shift+X) and install:

  • Python (by Microsoft) ‚Äì Must-have for Python development
  • Pylance ‚Äì Provides fast and smart IntelliSense (autocomplete)
  • Jupyter ‚Äì Enables Jupyter Notebooks for AI/ML work
  • Python Environment Manager ‚Äì Helps manage virtual environments
  • Live Share (optional) ‚Äì If collaborating with others
  • GitHub Copilot (AI Code Assistant) (optional) ‚Äì Improves autocomplete

Step 3: Set Up a Virtual Environment (Best Practice)

Using a virtual environment keeps dependencies isolated.

  1. Create a virtual environment: 
    python -m venv venv
  2. Activate it:
    • Windows (Command Prompt): 
      venv\Scripts\activate
    • Windows (PowerShell): 
      .\venv\Scripts\Activate.ps1
    • Mac/Linux: 
      source venv/bin/activate
  3. Confirm activation: 
    python -m pip install --upgrade pip

Step 4: Install AI & ML Libraries

Inside the virtual environment, install essential Python libraries:

pip install numpy pandas matplotlib seaborn scikit-learn jupyter notebook

For Deep Learning:

pip install tensorflow keras torch torchvision torchaudio

For NLP:

pip install nltk spacy transformers

For Data Handling:

pip install pandas openpyxl beautifulsoup4 requests

Step 5: Configure VS Code for Python

1. Open VS Code & Select the Python Interpreter: Press Ctrl+Shift+P ‚Üí Type "Python: Select Interpreter" Choose the Python version inside your virtual environment.

2. Enable Jupyter Notebook Support: Open a .ipynb file OR run: 

jupyter notebook

3. (Optional) Autoformat Code for Readability: Install black: 

pip install black
Enable format on save in VS Code: Go to Settings (Ctrl+,) Search for Format On Save and enable it.

Step 6: Configure VS Code Settings for Python

Recommended Settings (JSON format) Go to Settings (Ctrl+,) ‚Üí Click "Open Settings (JSON)" ‚Üí Add: 

                        
                            {
                                "python.formatting.provider": "black",
                                "editor.formatOnSave": true,
                                "editor.codeActionsOnSave": {
                                    "source.organizeImports": true
                                },
                                "python.analysis.typeCheckingMode": "basic",
                                "python.linting.enabled": true,
                                "python.linting.pylintEnabled": true
                            }

Step 7: Test Your Setup

Try running a simple script:

import tensorflow as tf rint("TensorFlow version:", tf.__version__)

If it prints the TensorFlow version, everything is set up correctly!

4. Run your first script

Let's create a simple Python script that prints Hello, World! to the console.

  1. Open your IDE or text editor and create a file named hello.py.
  2. Add this code: 
    
# hello.py
print("Hello, World!")
  3. Save the file and open a terminal.
  4. Navigate to the file's directory and run: 
    python hello.py
  5. You should see: 
    Hello, World!
